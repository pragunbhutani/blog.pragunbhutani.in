Google Summer of Code 2013 - Mobilize Wikidata

Written by Pragun Bhutani

It‘s with great excitement that I announce that I’ve been accepted into Google Summer of Code 2013. I'm going to be working with the Wikimedia foundation on a project entitled “Mobilize Wikidata”.

My project is aimed at extending Wikidata's functionality to make it accessible on mobile devices. Setting up Wikibase with MobileFrontend shows that while MobileFrontend can be used to achieve a mobile-friendly version of Wikidata, some problems arise due to JS based UIs not rendering properly through MobileFrontend.

My plan is to implement Wikibase without Javascript to make it compatible with MobileFrontend.

More details about the project can be found here.

I‘m really excited about the summer and all that it has to bring. This is my first time working with an organization and code base of this scale and I’m sure it's going to be a great experience.

This will be a good opportunity for me to learn how to enhance and improve upon existing code at such a huge scale. Mediawiki follows a lot of good coding conventions so I‘m hoping it’ll improve my coding style as well!

Anyway, stay tuned to this category and I'll keep reporting the progress of my project as it moves along.
