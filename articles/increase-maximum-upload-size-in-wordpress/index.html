Increase Maximum File Upload Size in Wordpress on DigitalOcean

Written by Pragun

I recently purchased a small instance on Digital Ocean because my mom wanted to set up a blog and I needed some place to try things out every now and then. I set up a simple Wordpress blog for her but everytime I would try to upload a theme, I would get a message saying that the uploaded file exceeded the maximum allowed size.

Note: My instance is running on Ubuntu 12.10 x64. 

The uploaded file exceeds the upload_max_filesize directive in php.ini.

I tried making my own php.ini in the wp-admin folder and editing the .htaccess file to allow bigger file sizes. Ultimately, I found the following fix to work just fine.

sudo nano /etc/php5/apache2/php.ini

And then look for the line which says upload_max_filesize = 2M and replace with:

upload_max_filesize = 64M
post_max_size = 64M
max_execution_time = 300

And that should do it for you!

Note : Depending upon who you're using as your hosting provider, there might be a different php.ini file that may be overriding the others to provide the max_upload_filesize directive and this method might not work for you. This has been tried and tested on only DigitalOcean.
