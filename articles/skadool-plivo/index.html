Skadool - Flexible scheduling powered by Plivo

Written by Pragun Bhutani

Plivo is a great cloud platform that allows you to build powerful telephony apps in a breeze. Now I'm not much of a developer, but when I was asked to use it and build an app, I decided to give it a spin! It was a little intimidating at first, but fortunately Plivo has very well documented code and a fantastic set of helper functions that made my job so much easier.

The idea for my app came from the need to make sure that people don‘t miss courier deliveries. I, for one, hate it when I’ve purchased something online and miss the shipment when there's noone there to receive it. Courier companies must not be very fond of it either, seeing as they have to then ship it all over again. Such a terrible waste of manpower hours.

How cool would it be if on the morning of your delivery, the courier-system could place an auto-generated IVR call or send a text to the delivery recipients asking them if they'd be available to receive their deliveries that day?

Skadool is my attempt to create such an app. I‘ve used Plivo’s framework to power it. I added a small API(my first, in fact!) to it to allow potential courier companies to handle the CRUD of this list.

Code and documentation can be found at https://github.com/blotmandroid/skadool.

I welcome any and all feedback! :)
